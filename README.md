## Интернет-магазин электроники ##

Данный проект - полноценный интернет-магазин для продажи электроники, смартфонов, компьютеров и комплектующих. Он включает полный каталог товаров с техническими характеристиками, корзину, избранное, отзывы, систему управления заказами и гибкую доставку (курьер или пункты выдачи). Разработан в качестве проекта-портфолио на производственной практике.

## Технический стек
- Python 3.11+
- Django 4.2
- Django REST Framework (DRF)
- PostgreSQL
- psycopg2-binary
- python-dotenv
- pytest / unittest
- Docker
- Docker Compose
- Git

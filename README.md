## Интернет-магазин электроники и бытовой техники ##

Данный проект - полноценный интернет-магазин для продажи электроники, бытовой техники, компьютеров и комплектующих. Он включает полный каталог товаров с техническими характеристиками, корзину, избранное, отзывы, систему управления заказами и гибкую доставку (курьер или пункты выдачи). Разработан в качестве проекта-портфолио на производственной практике.
